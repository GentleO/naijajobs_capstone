# NaijaJobs — Premium Nigerian Job Listing Platform 🇳🇬
(Frontend Track)

A high-performance, modern job board dedicated to linking exceptional Nigerian tech and creative professionals with top-tier domestic enterprises and global operations. This platform handles dynamic search routing, category filtering pipelines, user submission form with client-side validation, and local data persistence arrays.

This site is built as a Capstone Project for the **3MTT Software Development program (2026 [AirtelNextGen Cohort])**.

---

## 🎨 Visual System & Design Architecture

This project aligns with design instructions, avoiding external libraries (No Bootstrap, No Tailwind) to prove native structural mastery:

* **Color Profile:** Forest Green (`#1A5C38`), Nigerian Gold (`#F5A623`), Off-White Canvas (`#FAFAF8`), Dark Charcoal (`#1A1A1A`).
* **Typography Archetype:** Brand Titles configured in *Playfair Display* (Google Fonts); Functional interface text rendered under *DM Sans* (Google Fonts).
* **Layout Framework:** Responsive structural elements using CSS Grid for multi-card dashboard, combined with Flexbox alignment properties for  elements.
* **Theme Engine:** Native global selectors tracking Dark Mode toggling with complete operational context memory across page reloads.
